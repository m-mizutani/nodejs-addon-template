# module template of node.js

This is minimum set of node.js module.

## How to use

1. install npm

    `% curl https://npmjs.org/install.sh | sh`

2. install node-gyp

    `% npm install -g node-gyp`

3. run node-gyp

    `% node-gyp configure build`

4. run node.js

    `% node test.js`
